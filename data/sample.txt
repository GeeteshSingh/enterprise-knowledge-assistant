Apache Kafka is a distributed event streaming platform used for building real-time data pipelines and streaming applications.

Kafka stores data as a sequence of records called events. Each event has a key, value, and timestamp.

Kafka topics are categories to which records are published. Topics are split into partitions for scalability.

Kafka consumers read data from topics. Multiple consumers can form a consumer group to share the workload.

Kafka producers write data to topics. Producers can choose which partition to write to using a key.

Kafka brokers are servers that store and serve topic data. A Kafka cluster consists of multiple brokers for fault tolerance.

Apache Kafka integrates with Apache Spark for real-time stream processing and with Apache Flink for stateful computations.
